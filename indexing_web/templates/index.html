Cómo usar esta herramienta

      

        Pensada para tiendas Shopify (sitemap automático, blogs, productos y páginas),
        pero sirve para cualquier sitio web siempre que las URLs sean públicas
        (https://tudominio.com/...)
        y tengas configurado Google Search Console y Bing Webmaster Tools para ese dominio.
      


      
    
        
  1. 
          1
          
            Configura una sola vez en la pestaña
            Configurar APIs (o en las carpetas google_indexing/ y
            bing_indexing/): OAuth o cuenta de servicio de Google, clave IndexNow de Bing y el archivo
            TU_CLAVE.txt en la raíz de tu dominio (Shopify: Archivos + redirección URL).
          
        
    2. 
        
  2. 
          2
          
            Añade URLs: pégalas, sube un .txt/.csv de Search Console o pulsa
            Cargar sitemap (en Shopify suele ser /sitemap.xml).
          
        
    3. 
        
  3. 
          3
          
            Envía:
            Google (Indexing API),
            Bing (IndexNow) o
            Ambos. Revisa el panel de resultado (códigos 200/202 = recibido).
          
        
    4. 
        
  4. 
          4
          
            Comprueba en los paneles: Google Search Console y Bing Webmaster Tools.
            Esta app notifica cambios; no garantiza indexación inmediata.
          
        
    5. 
      


      

        Shopify: el sitemap se actualiza solo al publicar productos, colecciones o entradas de blog.
        Para páginas nuevas “descubiertas pero no indexadas”, conviene enviarlas aquí y enlazarlas desde el blog o la home.
        Cuotas: Google ~200 avisos/día; Bing por lotes (IndexNow). Uso masivo: reparte en varios días.
      


      

        Arrancar la app en tu PC:
        cd indexing_web → pip install -r requirements.txt → python app.py
        Luego abre http://127.0.0.1:5055. Solo funciona en tu equipo (no es pública en internet).
